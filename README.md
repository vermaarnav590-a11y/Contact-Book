# 📒 Contact Book

A command-line Contact Book application built using Python.

## Features

- Add Contact
- View Contacts
- Search Contact
- Update Contact
- Delete Contact
- Menu-Driven Interface

## Technologies Used

- Python 3

## How to Run

```bash
python main.py
```
